# Elpris Widget

Viser den faktiske danske elpris time for time for Brænderigården 7, 7500 Holstebro
på Gasel-aftalen "El til børspris + 6 øre" hos netselskab L-net (DK1).

## Indhold
- `CLAUDE.md` – projektkontekst og kodeprincipper (læses automatisk af Claude Code)
- `docs/elpris-spec.md` – fuld prisberegning, konstanter og kilder (sandheden)

## Kom i gang
Åbn mappen i Claude Code. Teknologivalget er bevidst ikke fastlagt endnu, se
"Åbne beslutninger" i `CLAUDE.md`.

## Datakilde
Spotpris fra elprisenligenu.dk (DK1). Øvrige tariffer er faste konstanter, se specten.
